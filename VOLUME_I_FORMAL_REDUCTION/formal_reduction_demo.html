Formal Reduction Demo – Parseval / Toeplitz Bridge

This demo numerically verifies the formal reduction: the same object appears as (i) an integral against a positive kernel and (ii) an arithmetic Toeplitz quadratic form.
We compute both and compare.

Parameters

Bandwidth of kernel (H > 0)
Truncation of Dirichlet polynomial
Height parameter
Real part of exponent (typically 0.5)
# integration points (odd, Simpson)
Integration window: [-range·H, range·H]

Definitions (conceptual)

D_N(T; σ) = Σ_{n=1}^N n^{-σ} e^{-i T log n}

w_H(t) = sech²(t / H)
g_{λ*}(t) = (6 / H²) sech⁴(t / H),  λ* = 4 / H²

ŵ_H(ω) = π H² ω / sinh(π H ω / 2)
ĝ_{λ*}(ω) = (ω² + λ*) ŵ_H(ω)

Integral form (Bochner side):
  F̃₂^{int}(T₀; H, N) = ∫ g_{λ*}(t) |D_N(T₀ + t)|² dt

Toeplitz form (arithmetic side):
  F̃₂^{toe}(T₀; H, N) =
    Σ_{m,n=1}^N (mn)^{-σ} (m/n)^{-iT₀} ĝ_{λ*}(log m - log n)

Parseval / convolution identity (formal reduction):
  F̃₂^{int}(T₀;H,N) ≡ F̃₂^{toe}(T₀;H,N)
